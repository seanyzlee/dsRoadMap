# dsRoadMap
Roadmap documenting my journey for a career in data science and ML  
  
Note for self: essentially, think of data science as the top of the iceberg. Once you learn data science fundamentals, you may start to branch off and specialize. For example, after getting some experience in data science, you may look into AI/ML (mid of the iceberg), and then deep learning (bottom of the iceberg). In no way is this comparison 100% accurate, but it's just here to paint an illustration.
